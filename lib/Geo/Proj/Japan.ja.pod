=head1 NAME

Geo::Proj::Japan - Geo::Projモジュールに日本の主な測地系定義を加えます


=head1 VERSION

この文書は Geo::Proj::Japan バージョン 0.0.2について記述しています。


=head1 SYNOPSIS

  use Geo::Proj::Japan;
  
  # 上記により、tokyo97、tokyo、jgd2000をGeo::Projで使えるようになります。
  my $point_tokyo = Geo::Point->latlong($lat, $lng, 'tokyo');


=head1 DESCRIPTION

このモジュールを導入する事により、Geo::Projで旧日本測地系（tokyo97）、
tokyo（tokyo97のエイリアス）、新日本測地系（jgd2000）をGeo::Projで
使えるようになります。

このモジュールは定義のみを提供し、関数・メソッド等は提供しません。


=head1 DEPENDENCIES

Geo::Proj


=head1 SEE ALSO

Geo::Proj
